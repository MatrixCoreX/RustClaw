# 你用 Claw 能做什么

用一句话说：**在 Telegram、WhatsApp 或网页里跟一个助手对话，让它帮你查东西、跑命令、发文件、看图听音、看行情，甚至执行多步任务，中途失败了还能说「继续」接着做。**

---

**查系统、跑命令**  
问当前目录有没有隐藏文件、执行 `df -h` 看磁盘、跑 `pwd` / `whoami`、数文件、看进程、重启服务等，直接说需求，它会选命令并执行。

**搜文件、看日志**  
按名字或内容搜文件、列目录、看压缩包里的内容；把日志给它，让它帮你分析报错或异常。

**读写文件、发给你**  
让它写一段内容到指定文件、根据当前路径生成一句短诗保存成文件，或「把 readme.md 发给我」—— 它会发文件而不是贴一大段文字。

**看图、生成图、编辑图**  
发一张图问「这里有什么」；或说「生成一张…」「把这张图里的 XX 改成 YY」。支持图片理解、生成和编辑。

**语音转文字、文字转语音**  
发一段语音让它转成文字；或让它把一段文字读出来（TTS）。

**闲聊、写短句**  
讲个冷笑话、根据当前路径写一句打工人短诗、用大白话解释某个命令的输出，都可以。

**看行情、下单（需先配置 API）**  
查币价、看指标、看链上数据；配置好交易所 API 后可以预览下单、确认下单、查订单、撤单、看持仓。（凭证只存在你这台机器，按你的 key 区分。）

**看 RSS、抓网页**  
订阅 RSS、按关键词抓新闻或网页内容。

**Git、Docker、数据库**  
简单的 Git 操作、容器状态与启停、对本地数据库的受控查询/修改（在安全策略允许范围内）。

**多步任务 + 失败后继续**  
可以说「先执行 A，再执行 B，然后把结果用一句话总结」。它会按顺序执行；如果某一步失败了，你可以说「继续」只做剩余步骤，或问「刚才哪步失败了、还剩什么」。

---

你可以通过 **Telegram Bot**、**WhatsApp** 或 **网页端** 和它对话；同一套身份（key）下，记忆和权限在通道间一致。  
一句话：**能说的，都可以试着让它做。**
