# Consolidated singleton manual cases.
#
# Background: under cases/ there used to live ~18 single-case .txt files
# (nl_case_*_only.txt etc) created ad-hoc when triaging a specific bug.
# None of them were referenced by any script — they just leaked into the
# directory. This file consolidates them into one with the same format
# the runner already understands.
#
# The original singleton files are kept on disk with a deprecation header
# pointing here; you can safely delete them in a future cleanup once nobody
# is paste-running their paths anymore.
#
# Format: suite|name|tags|prompt|expect=<substring>
