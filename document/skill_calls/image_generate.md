# image_generate

- 脚本：`scripts/skill_calls/call_image_generate.sh`
- 默认参数：`{"prompt":"A minimal crab icon","size":"1024x1024"}`
- 示例：
  - `bash scripts/skill_calls/call_image_generate.sh`
  - `bash scripts/skill_calls/call_image_generate.sh --args '{"prompt":"A cyberpunk city","size":"1024x1024"}'`
- 常用参数：`prompt`, `size`, `style`, `quality`, `n`, `output_path`
