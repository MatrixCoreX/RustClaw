# package_manager

- 脚本：`scripts/skill_calls/call_package_manager.sh`
- 默认参数：`{"action":"info","manager":"npm","package":"react"}`
- 示例：
  - `bash scripts/skill_calls/call_package_manager.sh`
  - `bash scripts/skill_calls/call_package_manager.sh --args '{"action":"search","manager":"npm","package":"axios"}'`
- 常用参数：`action`, `manager`, `package`
