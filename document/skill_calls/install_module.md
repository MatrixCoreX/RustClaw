# install_module

- 脚本：`scripts/skill_calls/call_install_module.sh`
- 默认参数：`{"manager":"npm","module":"lodash","dry_run":true}`
- 示例：
  - `bash scripts/skill_calls/call_install_module.sh`
  - `bash scripts/skill_calls/call_install_module.sh --args '{"manager":"pip","module":"requests","dry_run":true}'`
- 常用参数：`manager`, `module`, `version`, `dry_run`
