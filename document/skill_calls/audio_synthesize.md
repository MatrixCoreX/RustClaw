# audio_synthesize

- 脚本：`scripts/skill_calls/call_audio_synthesize.sh`
- 默认参数：`{"text":"你好，这是 RustClaw","voice":"nova"}`
- 示例：
  - `bash scripts/skill_calls/call_audio_synthesize.sh`
  - `bash scripts/skill_calls/call_audio_synthesize.sh --args '{"text":"Hello world","voice":"alloy"}'`
- 常用参数：`text`, `voice`, `format`, `output_path`
